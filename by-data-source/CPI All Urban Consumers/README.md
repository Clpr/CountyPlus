# README - CPI

## I. Pipeline

* **Input**: FRED code `CPIAUCSL`, annually average, seasonally adjusted
    * `CPIAUCSL.xls`
* **Processing files**:
    * `make.do`
* **Output**: 
    * `cpi.dta`
* **Variables**
    * `cpi`: CPI All Urban Consumers (1982-1984=100)
    * `year`: Year

## II. Data download

* Go to FRED, locate the index by its code, then download the series.
